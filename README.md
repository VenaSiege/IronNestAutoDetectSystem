# 铁巢自动化侦察系统

一个基于 `Python + Tkinter` 的本地桌面地图工具，用于铁巢自动化侦察、地图标识管理与目标定位计算。

## 运行方式

确保本机可用 `Python 3`，然后在项目目录执行：

```bash
python main.py
```

## 功能概览

- 20 x 10 大地图，列为 `A-T`，行为 `1-10`
- 单个大格支持进入 10 x 10 小格视图
- 固定比例尺：大格边长代表 `1km`，小格边长代表 `0.1km`
- 支持放置铁巢、观测点、参考点、敌军目标点
- 支持手动新建、打开、保存、另存为
- 支持在大地图摘要显示主要标识信息
- 支持“目标定位计算”：
  - 三角测距
  - 圆弧交叉测距
  - 混合测距
  - 参考点自动计算
